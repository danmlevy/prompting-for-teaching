# Prompting for AI in Teaching: GitHub Pages Publishing Package

Files:

- `index.html` — the interactive pre-work page.

To embed the PingPong bot, open `index.html` and find this block:

```html
<div class="bot-placeholder" id="bot-embed-location">
  <strong>Embed Prompt Coach bot here</strong>
  <p class="small">Replace this box with the PingPong embed code for the AI Teaching Prompt Coach.</p>
</div>
```

Replace the entire block with the PingPong embed code.
